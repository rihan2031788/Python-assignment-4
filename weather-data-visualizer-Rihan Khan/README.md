# Weather Data Visualizer

## Dataset
- File: `data/weatherIndia.csv`
- Source: Custom generated for India (Jan 2024)
- Columns: Formatted Date, Temperature (C), Humidity, Precip Type, Daily Summary

## Tools Used
- Python 3.14+
- Pandas, NumPy, Matplotlib

## Tasks Completed
1. Data Acquisition & Loading
2. Data Cleaning & Processing
3. Statistical Analysis (Daily/Monthly/Yearly)
4. Visualization (Line, Bar, Scatter, Combined Plots)
5. Grouping & Aggregation
6. Export & Reporting

## Output Files
- `cleaned_data.csv`: Processed dataset
- `plots/`: All PNG visualizations
- `summary_report.txt`: Insights and anomalies

## How to Run
```bash
pip install pandas numpy matplotlib
python weather_analysis.py
